GLOSSARY OF TERMS

Published: 21 MAR 2025

Terms

\(\forall\)
Universal quantifier: \(\forall x : A, P(x)\). See Forall.
\(\exists\)
Existential quantifier: \(\exists x : A, P(x)\). See Exists.
\(\text{Set}\)
Type of sets over a base type: \(\text{Set}(A) = \{ x : A \mid P(x) \}\). See Set.
\(\text{SetEq}\)
Set equality: \(S =_{\text{Set}(A)} T \iff \forall x : A, S(x) \iff T(x)\). See SetEq.
\(\mu\)
Measure constructor: \(\mu(A, \Sigma, f) : \text{Measure}(A, \Sigma)\). See Mu.
\(\text{Measure}\)
Measure type: \(\text{Measure}(A, \Sigma)\) for measurable spaces. See Measure.
\(\text{Seq}\)
Sequence type: \(\text{Seq}(A) = \mathbb{N} \rightarrow A\). See Seq.
\(\sup\)
Supremum: \(\sup S = \text{least upper bound of } S \subseteq \mathbb{R}\). See Sup.
\(\inf\)
Infimum: \(\inf S = \text{greatest lower bound of } S \subseteq \mathbb{R}\). See Inf.
\(\text{Lebesgue}\)
Lebesgue integral: \(\int_S f \, d\mu\). See Lebesgue.
\(\text{Limit}\)
Sequence limit: \(\lim_{n \to \infty} a_n = l\). See Limit.
\(\bigcup\)
Infinite union: \(\bigcup_{i : I} S_i\). See Union.
\(\bigcap\)
Intersection: \(S \cap T = \{ x \mid S(x) \land T(x) \}\). See Intersect.
\(\setminus\)
Complement: \(A \setminus S = \{ x : A \mid \neg S(x) \}\). See Complement.
\(\overline{S}\)
Closure: \(\overline{S} = S \cup \{ \text{limit points of } S \}\). See Closure.
\(\sum\)
Infinite sum: \(\sum_{i : I} a_i = \lim_{n} \sum_{k=0}^{n-1} a_k\). See Sum.
\(\land\)
Logical conjunction: \(P \land Q\). See And.
\(\lor\)
Logical disjunction: \(P \lor Q\). See Or.
\(\text{Prop}\)
Propositional universe: \(\text{Prop} = \{ P \mid P \text{ is 0-truncated} \}\). See Prop.
\(\mathcal{P}\)
Power set: \(\mathcal{P}(A) = \text{Set}(\text{Set}(A))\). See Power.
\(U_0, U_1\)
Type universes: \(U_0 : U_1\). See Universe.
\(\mathbb{R}\)
Real numbers: \(\mathbb{R} : U_0\) with limit axioms. See Real.
